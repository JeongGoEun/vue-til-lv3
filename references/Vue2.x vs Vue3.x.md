Vue2.x VS Vue3.x

- 명령어
    - 2.x : vue init '프로젝트 템플릿 이름' 파일 위치
    - 3.x : vue create '프로젝트 이름'

- 웹팩 설정 파일
    - 2.x : 노출 O
    - 3.x : 노출 X
        - 사용자들에게 노출시키지 않고 내부에서 알아서 처리하겠다
        - 추가하기 위해선 별도로 설정해줘야 함(vue-config.js 파일 활용)

- 프로젝트 구성
    - 2.x : 깃헙의 템플릿 다운로드(정해진 템플릿만 쓴다)
    - 3.x : 플러그인 기반으로 기능 추가

- ES6 이해도
    - 2.x : 불필요
    - 3.x : 필요
